Componentes

Los SIG se componen de 5 elementos mínimos que hacen llamarlo un sistema:

a) Datos

Información geoespacial: se caracteriza por ser una modelación de la realidad a través de vectores geográficos (puntos, lineas o polígonos) o a través de ráster (matriz de píxeles). Estas fuentes se caracterizan por almacenar información de la realidad con una explicita composición espacial, es decir está posicionado en alguna parte del mundo (localización).

b) Métodos

Corresponden a todas las técnicas y procesamientos que se le aplican a los datos para consultar información contenida, para alterar o transformar los datos para obtener información espacial nueva.

c) Software

Aplicación informática que permite la visualización y gestión de los datos espaciales. Comúnmente suele confundirse con que los SIG son solamente este componente.

d) Hardware

Corresponde al soporte informático que permite la apertura de un software y los datos. Este elemento puede ser un computador, un dispositivo móvil, etc.

e) Personas

Hace referencia tanto a las y los encargados de diseñar y mantener el SIG como a los usuarios que utilizan la información geoespacial.
